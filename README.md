BSP-Tree Demonstration
======================
Just a Demonstration shows how BSP-Tree works in visualization.

Start
=====
Download the BSPTree.jar and open it.

Control
=======
* mouse left - drag to draw.
* mouse right - locate where the cursor is belowed to.
* space - more command.
* r - reset.
